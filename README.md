Credit Card Fraud Detection

Machine learning project to detect fraudulent credit card transactions using Random Forest and XGBoost classifiers.

Dataset
284,807 real credit card transactions with 492 fraud cases (0.17% fraud rate) from Kaggle.

Models Used
Random Forest (AUC: 0.9804)
XGBoost (AUC: 0.9724)

Key Results
XGBoost achieved 87% precision and 84% recall on fraud detection
V14 was the single most important feature for identifying fraud
Both models vastly outperformed random baseline

Tech Stack
Python, Pandas, Scikit-learn, XGBoost, Matplotlib, Seaborn, Google Colab
